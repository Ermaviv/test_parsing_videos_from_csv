# Отчет о потреблении кофе

## Назначение
Приложение читает файлы с данными о видео
и формирует отчет.

## Как использовать
- Скачать папку с данными (кодом и пр. файлами)
- Установить виртуальное окружение
```shell
python -v venv venv
```
- Активировать виртуальное окружение
```shell
./venv/source/activate
```
- Установить зависимости (желательно, в новом виртуальном окружении)
```shell
pip install -r requirements.txt
```
- Запустить сервис
```shell
python main.py --files <указать_входные_файлы> --report <установить название файла для отчета>
```
### Подробнее о входных параметрах
#### `<указать_входные_файлы>`
Здесь указываются пути к файлам или названия файлов
(если они лежат в той же директории, что сама программа),
в которых содержится перечень информации о разных видео.
Сами файл(ы) должны иметь шапку (headers) следующего типа:
`title,ctr,retention_rate,views,likes,avg_watch_time`
- title - название видео
- ctr - click-through rate, кликабельность (процент пользователей, которые нажали на превью после того, как увидели его)
- retention_rate - удержание (процент видео, который досматривают в среднем)
#### `<установить название файла для отчета>`
Указывается полное название (вместе с расширением) отчетного файла.
